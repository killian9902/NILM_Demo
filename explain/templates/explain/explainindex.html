AI Explained

We have created our own in-house AI model, designed specifically for NILM.

Using state-of-the-art deep learning methods we have produced a model that recognises appliance-specifc patterns. The pattern of an appliance can be inferred by frequency of load, variance, signature load, etc.

The model then recognises these patterns amongst other underlying trends that may have seasonal and cyclical association, to detect 1. which appliances are being used and 2. how long they are being used for. There are many possible underlying trends: devices being used together (coffee machine and toaster), time association (TV on at night), increase conumption (new person in house), etc.

Deep learning models contain multiple processing layers, each designed to analyze the data and transform it for the next layer. These layers are interconnected, and as the data moves from layer to layer, more complex patterns are uncovered. The first layers detect basic patterns, and the last layers detect small changes in the data that would otherwise go unnoticed. Every time the model receives new example data, these layers update to become more accurate.
